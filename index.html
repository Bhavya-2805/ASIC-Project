01 / Problem Statement

The Challenge

Automotive safety systems require deterministic, low-latency responses. AI/ML approaches are too resource-heavy for embedded ASIC deployment. We need pure hardware logic.

😴

Driver Fatigue

Drowsiness accounts for ~20% of fatal accidents. Detecting blink frequency and head drop events provides early warning before impairment is critical.

🍺

Alcohol Impairment

8-bit ADC output from a breath/skin alcohol sensor is compared against a threshold (128/255). Above threshold triggers immediate CRITICAL alert.

🚗

Lane Deviation

Lane offset computed by an external vision system is fed as an 8-bit value. Deviation above 100/255 triggers WARNING or contributes to DANGER escalation.

⚡

Why ASIC?

No OS, no scheduler, no latency jitter. Pure hardware finite state logic runs deterministically every clock cycle at 50 MHz — 20 ns response time.

02 / Architecture

System Architecture

Five independent modules connected through a priority-based alert controller. All signals are synchronised to the rising edge of the 50 MHz clock.

04 / Design Flow

RTL to GDSII

Complete open-source ASIC design flow from behavioural RTL coding through to a fabrication-ready GDSII layout file.

1
DESIGN

RTL Coding

Behavioural Verilog HDL. Synchronous design with active-low reset, non-blocking assignments, and modular sub-module architecture.

▸ Icarus Verilog · GTKWave
2
VERIFICATION

Functional Simulation

Testbench with manual stimuli for all four alert states. GTKWave waveforms confirmed correct FSM transitions — SAFE → WARNING → DANGER → CRITICAL.

▸ iverilog · vvp · GTKWave
✓ All states verified
3
SYNTHESIS

Gate-Level Mapping

Yosys synthesises RTL into the OSU035 standard cell library. Counters become adder chains + flip-flops; comparators map to logic gate networks. 64 cells, 51 wires.

▸ Yosys · osu035
✓ 78 cells · ~200 GE
4
PHYSICAL

Placement & Routing

Graywolf places 78 standard cells in optimised rows. Qrouter connects all nets using Metal1/Metal2/Metal3 layers. No unrouted nets. Clock inserted with CLKBUF1 buffers.

▸ Graywolf · Qrouter · Magic
✓ 96 nets fully routed
5
SIGNOFF

DRC · LVS · STA

DRC passed with zero violations. LVS confirmed layout-to-netlist equivalence. STA computed max clock frequency of 1655 MHz — far exceeding the 50 MHz constraint. Max path delay: 604 ps.

▸ Magic · Netgen · OpenSTA
✓ DRC: 0 errors · LVS: matched
6
TAPE-OUT

GDSII Generation

Final GDSII file generated via KLayout. Each layer — diffusion, polysilicon, Metal1–Metal3 — encoded as geometric shapes. Foundry-ready binary layout file produced.

▸ Qflow gdsii · KLayout
✓ driver_top.gds — READY
